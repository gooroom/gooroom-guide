Gooroom Platform Security Framework
Gooroom security framework is a special security enhancement device unique to the Gooroom Platform for high security level.
Trusted Booting

Detects the system tampering
by security intrusion(hacking)
and prevents booting when
booting Gooroom OS.

Protect Executable Files

By checking validity
using self verification key,
prevents the forgery and
tampering of the main modules and
executable files.

Resources Control

Controls access permission
to the external storage media,
printer, network, etc of the user
who logged in to Gooroom OS.

Protecting OS

Through OS self-protection
technology, monitors and blocks
of internal data corruption
from system malware.
